
# 📚 Sistema de Matrícula Escolar

Projeto desenvolvido para simular um **sistema de matrícula para uma rede estudantil**, utilizando **CRUD** na linguagem **Java** e integração com **banco de dados MariaDB**.

## 🚀 Tecnologias utilizadas

- Java
- Apache NetBeans
- MariaDB
- Maven

## ⚙️ Funcionalidades

- Cadastro de alunos
- Matrícula de estudantes
- Consulta de registros
- Atualização e remoção de dados (CRUD)
